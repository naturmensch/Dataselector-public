# Thesis Decision Evidence

This directory is the stable repo-side home for the tracked thesis decision
evidence that is still referenced by active policy and config contracts.

Status:
- active reference evidence
- secondary to the canonical thesis runbook
- not the default user entrypoint

Use this directory for:
- min-distance pre-registration and decision artifacts
- `n_samples` pre-registration and decision artifacts
- the tracked summary CSV/Markdown evidence that backs those decisions

Do not confuse this area with:
- `docs/reports/`, which is historical/generated reporting output
- `docs/07_ARCHIVE/`, which is repo-facing historical documentation
- `outputs/`, which is the non-versioned run/output root

If you want to run the thesis workflow, start with:
- `README.md`
- `docs/INDEX.md`
- `docs/03_USER_GUIDES/THESIS_PIPELINE_HOWTO.md`
