# Glossary

Short glossary of key acronyms and terms:

- KDR100: Karte des Deutschen Reiches (100 sheet series)
- LHS: Latin Hypercube Sampling
- QMC: Quasi-Monte Carlo (Sobol)
- UQ: Uncertainty Quantification
- UMAP: Uniform Manifold Approximation and Projection
- DINOv2: Self-supervised ViT-based feature extractor
- Apricot: Submodular selection library for Facility Location
