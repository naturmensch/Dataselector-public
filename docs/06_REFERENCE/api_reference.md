# API Reference (Overview)

Compact reference of primary public APIs in the current package structure.

## Core APIs

1. `dataselector.features.feature_extractor.FeatureExtractor`
2. `dataselector.selection.diversity_selector.DiversitySelector`
3. `dataselector.pipeline.experiment_manager.ExperimentManager`
4. `dataselector.workflows.thesis_pipeline.run_thesis_pipeline`

## CLI Entry Points

Use:

```bash
python -m dataselector <command>
```

For the thesis production path:

```bash
python -m dataselector thesis-pipeline
```

Legacy `src/*` import paths are non-authoritative.
